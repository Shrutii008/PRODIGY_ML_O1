House Price Prediction

Overview

This project predicts house prices using Machine Learning and Linear Regression.

Dataset

The project uses the House Prices dataset from Kaggle.

Technologies Used

- Python
- Pandas
- Scikit-learn

Features

- Loads housing data from a CSV file
- Trains a Linear Regression model
- Predicts house prices
- Calculates Mean Absolute Error (MAE)

How to Run

1. Install the required libraries:
   - pandas
   - scikit-learn
2. Place "train.csv" in the project folder.
3. Run the following command:

python main.py
